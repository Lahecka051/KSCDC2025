# STL — 3D Print Models

KSCDC2025 프로젝트에서 자체 설계 후 3D 프린팅한 부품 모델 모음.

## Drone/

드론 본체에 장착되는 부품. 소화볼(4인치)을 거치/투하하기 위한 BAT(Bomb Attachment) 하우징과 플레이트.

| 파일 | 용도 |
|------|------|
| `Drone_4inch_Ball.stl` | 4인치 소화볼 검증용 더미 모델 |
| `Drone_BAT_House_Upper.stl` | 소화볼 하우징 상부 |
| `Drone_BAT_House_Lower.stl` | 소화볼 하우징 하부 |
| `Drone_BAT_House_Hook.stl` | 소화볼 고정 후크 (서보 회전으로 해제) |
| `Drone_Bomb_Plate.stl` | 드론 하단 BAT 마운트 플레이트 |

## Station/

스테이션(자동 장전기)의 랙앤피니언 메커니즘 부품. 개발 중단(WIP) 상태.

| 파일 | 용도 |
|------|------|
| `Station_Rack.stl` | 직선 이송용 랙 기어 |
| `Station_Pinion.stl` | 서보 결합 피니언 기어 |
| `Station_Bar.stl` | 가이드 바 |

## 출력 권장 사양

| 항목 | 값 |
|------|-----|
| 필라멘트 | PLA (자세한 사양은 [Datasheets/Filament.md](../Datasheets/Filament.md)) |
| 레이어 두께 | 0.2 mm |
| 인필 | 30 ~ 50% (하중 부품은 50%) |
| 서포트 | 부품별 필요 시 |
